# BAD.AI Solutions

A simple, modern HTML template showcasing BAD.AI services and projects.

This project is built using:
- **HTML5**: The structure of the website.
- **Tailwind CSS (via CDN)**: For rapid UI development and styling.
- **Google Fonts** (`Poppins` and `Raleway`): Custom typography.

## Getting Started

Follow these steps to explore or modify the project on your local machine.

### Prerequisites

You need a web browser (like Chrome, Firefox, or Safari) to view this project. No additional server setup is required out of the box.

### Running Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/anfieldlad/badai-tech.git
   ```
2. **Navigate into the project directory:**
   ```bash
   cd badai-tech
   ```
3. **Open the project:**
   Simply double-click the `index.html` file in your file explorer to open it in your default web browser.

   Alternatively, if you prefer a live-reloading development server, you can use a tool like [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) in VS Code or run it with Python:
   ```bash
   python -m http.server 8000
   ```
   Then point your browser to `http://localhost:8000`.

## Features
- **Responsive Navigation**: Includes a mobile-friendly hamburger menu.
- **Smooth Scrolling**: Navigate sections elegantly without page reloads.
- **Styling**: Pre-configured with Tailwind CSS colors, typography, and hover effects.
